
# MapleSight

MapleSight is a browser-based image-processing web application built with OpenCV.js.

Hosted site: https://maplesight.github.io/MapleSight/

Usage

- Open the hosted site URL.
- Click Select Screen and select the Maplestory window.
- Move Buff Favorites to just above the HP bar.

## Current features

- Dawn Notifer : Detects if Dawn is present in the buff favorites and will beep once it is unable to find the buff Icon.
